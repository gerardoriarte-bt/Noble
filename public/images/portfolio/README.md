## Especificaciones Técnicas

- **Formato**: JPG o PNG
- **Resolución**: Mínimo 1200x800px
- **Estilo**: Arquitectura minimalista, espacios limpios
- **Colores**: Preferiblemente tonos neutros (grises, blancos, beiges)
- **Calidad**: Alta resolución para web

## Estilo Visual

Las imágenes deben reflejar:
- Arquitectura minimalista y moderna
- Espacios limpios y ordenados
- Uso de luz natural
- Materiales naturales (piedra, madera, concreto)
- Composición geométrica simple
